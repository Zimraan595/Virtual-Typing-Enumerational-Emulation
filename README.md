# re-type

## What It Does

re-type is a simple tool that gives you a smarter way to paste text. Instead of pressing `Ctrl+V` to instantly paste everything at once, you press `Ctrl+Q` and re-type will type out your copied text for you — one character at a time, just like a real person would.

To make it look even more natural, re-type adds small random delays between each keystroke so the speed isn't perfectly constant. It even makes the occasional typo on purpose and immediately corrects it with backspace, just like a human would. The end result is text that looks like it was genuinely typed out by hand.

## Getting Started

1. Download the latest release `.zip` file.
2. Unzip/extract the folder.
3. Double-click `re-type.exe` to run it.

That's it — no installation required.

## How to Use It

1. Copy any text to your clipboard like you normally would (`Ctrl+C`).
2. Open re-type by double-clicking the `.exe`.
3. Click into whatever text field or document you want the text typed into.
4. Press `Ctrl+Q` to start typing.

re-type will begin typing out your clipboard text character by character. You can use it in any app, text box, or document — anywhere you can normally type.

## Stopping Mid-Type

If re-type is in the middle of typing and you want it to stop, press `Ctrl+Shift+Q` and it will immediately cancel.

## Changing the Speed

When you first launch re-type, it will ask if you want to change your settings. Type `yes` and hit Enter to adjust the typing speed.

The speed setting is a multiplier:

- `1` = normal speed
- `2` = twice as fast
- `5` = five times faster
- `0.5` = half speed (slower, more deliberate)

Your setting is saved automatically, so you won't have to set it again next time unless you want to change it. Just type `no` when asked on future launches to keep your current speed.

## Modifier Key Safety

If you hold down `Ctrl`, `Shift`, or `Alt` while re-type is typing, it will automatically pause until you let go. This prevents it from accidentally triggering shortcuts or messing up what it's doing while you're pressing keys.

## Features at a Glance

- **Clipboard typing** — copies your clipboard and types it out character by character
- **Variable typing speed** — random delays between keystrokes so it doesn't look robotic
- **Simulated typos** — occasionally presses a wrong key and corrects it, mimicking real typing
- **Adjustable speed** — set a speed multiplier to make it faster or slower
- **Cancel anytime** — press `Ctrl+Shift+Q` to stop typing immediately
- **Modifier key pausing** — automatically pauses if you hold Ctrl, Shift, or Alt
- **Persistent settings** — your speed preference is saved between sessions
